day11 เริ่มจะชิน(ชา)

เริ่มต้นสัปดาห์ที่สามของการฝึกงาน ผมได้โจทย์ให้ไปลองศึกษา NAS (Network attached server) ตัวที่เลือกมาลองคือ FreeNAS ตัวมันเองเป็น Open source การคอนฟิกหลังจากจัดการ assign IP address แล้วที่เหลือทำผ่าน web UI ได้เลย

ตอนบ่ายพี่ CEO เรียกตรวจ Resume อีกรอบ ได้รู้จัก Tag line เพื่อบรรยายตัวเองให้ได้ในหนึ่งบรรทัด

ตกเย็นไม่ได้แวะที่ไหน ตรงดิ่งกลับบ้านเพราะอ่านหนังสือที่ซื้อมาจากงานหนังสือ
