รีวิวหูฟัง HyperX Cloud Core

Cloud Core เป็นหูฟังตัวล่างสุดในซีรีย์เกมมิ่งจาก Hyper X ที่ตัดลูกเล่นพิเศษออกทั้งหมดทำให้มีราคาลดลงมาด้วย

การออกแบบ

หูฟังใช้อลูมิเนียมสีดำด้านเป็นวัสดุหลัก ฟองน้ำและก้านด้านบนเป็นหนังเทียมแบบ Memory Form ปรับขนาดได้ทิศทางเดียวคือด้านบน-ล่าง กันเสียงรอบข้างได้ดี

ความสบาย

เป็นหัวข้อที่ผมลังเลมากเพราะมีทั้งกลุ่มที่คนใส่แล้วบอกว่าสบายส่วนมากจะเป็นชาวต่างชาติ และอีกกลุ่มที่บอกว่าหูฟังบีบแรงทำให้ใส่ต่อกันนานๆ ไม่ได้ ดังนั้นส่วนนี้ผมอยากให้ไปทดลองเองที่ร้านจะมั่นใจที่สุด

เสียง

หูฟังใช้ไดรเวอร์แบบ Closed-back ขนาด 53mm ที่ถือว่าใหญ่กว่าหูฟังในช่วงราคาเดียวกัน โทนเสียงจะออกไปด้านกลาง-ทุ้ม ผมเข้าใจว่าจูนเสียงมาเพื่อเล่นเกมทำให้ฟังเพลงไม่เพราะเสียเลย เสียงทุ้มลงต่ำได้พอสมควรแต่ลูกใหญ่ไม่ชัดเจน เสียงกลางเด่นและคมมาก เสียงสูงไปไม่สุดจะเหลือแค่เสียงแหบๆ

ไมโครโฟน

สามารถถอดได้แต่พับเก็บไม่ได้ ตำแหน่งรับเสียงยื่นเข้ามาที่ปาก ทำให้รับเสีียงหายใจไปด้วย เสียงที่ได้จะออกแห้งแต่ชัดเจนสำหรับการสนทนา

ข้อเสีย

  • ไม่มี Control panel ในสายให้ปรับเสียงหรือปิดไมค์
  • วัสดุเป็นหนัง บางคนอาจรู้สึกร้อน
