day16 นับตลอด นับ

จากวันก่อนๆ ได้ศึกษาการทำ Mail server ด้วย Postfix วันนี้ก็ได้เจอกับอีกหนึ่งวิธีในการส่งอีเมล์โดยไม่จำเป็นต้องตั้ง Mail server และไม่ต้องปวดหัวกับเรื่อง IP blacklist (เกิดจากถ้า IP address หนึ่งส่งสแปมออกไปเยอะจนโดน Blacklist ทำให้เมลที่ออกจาก IP address นี้วิ่งเข้า Junk) คือการใช้บริการจาก Mail sender ทั้งหลายเช่น Mailgun, AWS SES เป็นต้น

ก่อนกลับบ้านได้งาน Cross-check ว่าภายในห้อง IDC มีเครื่องประเภท Bit-Torrent colocation อยู่กี่เครื่อง เปิดใช้งานกี่เครื่อง ระงับบริการกี่เครื่อง รอนำออกกี่เครื่อง เพื่อเช็คยอดกับฐานข้อมูล
